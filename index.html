What this tool does, and what it does not do

It replaces NHS numbers with stable Patient IDs (Patient 0001, 0002…) — the same NHS number always becomes the same Patient ID, so you can join multiple extracts on the Patient ID. It detects NHS numbers both by column header and by validating the cell contents against the NHS Modulus-11 check digit, so an unrecognised header (e.g. "NHS #") no longer lets numbers slip through. It redacts whole columns whose headers match common patient-identifier patterns (name, DOB, address, postcode, phone, email).

  • It does not anonymise free-text fields ("Notes" columns may still contain identifiers written in by clinicians).
  • It does not anonymise PDFs, screenshots or images.
  • It does not protect against re-identification from very small cohorts with rare attribute combinations.

For cohorts under about 10 patients, treat the output as still identifiable and apply the same handling as the source.
